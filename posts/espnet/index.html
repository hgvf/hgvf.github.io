安裝 Kaldi

  1. Git clone Kaldi
$ cd <要安裝的位置>
$ git clone https://github.com/kaldi-asr/kaldi
  1. Install tools
$ cd <kaldi-root>/tools
$ make -j <NUM_CPU>

(在 lab 時候 <NUM_CPU> 可以用 32。)

  1. 使用 OpenBLAS 安裝 BLAS libraries
$ cd <kaldi-root>/tools
$ ./extras/install_openblas.sh
  1. 檢查有沒有 tools 缺漏
$ ./extras/check_dependencies.sh

(有缺漏就按照指示裝，沒有的話會顯示 all ok)

  1. Compile & install Kaldi
$ cd <kaldi-root>/src
$ ./configure --use-cuda=no
$ make -j clean depend
$ make -j <NUM_CPU>
